File System Analyzer

This command line tool will scan all the files and subfolders of the provided starting folder, extracts information and generate useful reports that should help better organize the file system. Currently supported:

  • Duplicates: files and folders. Hash (sha1) based
  • Similar folders.

Requirements:

Java 1.8

Usage:

Update provided settings.properties with the proper values:

Available options:

  • watchDirectory: Folder to analyze, value: valid path to directory
  • recurse: whether to go deeper that the provided folder, recommended value: yes
  • commandsRowOperations: When the report is build, what command line associate the paths found
  • skipProcessed: whether to ignore or not items already analyzed
  • commandsOperationSkipRow: whether to leave the last|first row on each group without command operation
  • showBiggestItems: how many items to show in the report

  • locations options:

    • logFolder=logs
    • duplicatesReport=duplicates.log
    • similarFoldersReport=similar_folders.log

Run:

java -Djava.library.path="lib" -jar "FileSystemAnalyserEclipse.jar"
